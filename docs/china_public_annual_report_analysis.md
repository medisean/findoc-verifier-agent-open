# 中国公开年报扩展样本分析

上市公司年度报告属于公开披露材料，可以作为 FinDoc Verifier Agent 的高质量验证样本来源。
本仓库保留公开来源入口、样本价值分析、推荐任务配置和 4 份正式 PDF。这样既能保证
可复核性，也能让本地页段验证和远端全量验证使用同一批输入。

## 关键观察

### 比亚迪

年报披露，比亚迪 2025 年实现营业收入约 8,039.64 亿元，同比增长约 3.46%；
归属于上市公司股东的净利润约 326.19 亿元。汽车、汽车相关产品及其他产品业务收入约
6,486.46 亿元，手机部件、组装及其他产品业务收入约 1,552.37 亿元。

适合验证的能力点：

- 在百页级 PDF 中定位主要会计数据、合并利润表、分部收入和研发投入。
- 把“亿元”“百万元”“人民币元”等单位统一到目标 schema。
- 校验分部收入与总收入的比例关系，识别四舍五入差异。
- 处理董事长致辞、业务讨论和财务报表之间的重复数字。

### 宁德时代

宁德时代官方公告披露，2025 年公司实现营业收入 4,237 亿元，同比增长 17%；归属于上市公司
股东的净利润 722 亿元，同比增长 42%；经营活动现金流量净额 1,332 亿元，研发投入 221 亿元。
同一公告还包含全年锂电池销量 661GWh、动力电池市占率 39.2%、储能电池出货量市占率 30.4%
等经营指标。

适合验证的能力点：

- 同时抽取财务指标和经营指标，并保留字段类型差异。
- 处理 `GWh`、百分比、现金流和研发投入等多种单位。
- 对“同比增长”“市占率提升”等文本关系建立 evidence。
- 区分年度报告正文摘要与正式报表页，避免把宣传性表述当作财务科目。

### 贵州茅台

贵州茅台 2025 年年度报告披露，营业收入为 1,688.38 亿元，同比下降 1.21%；归属于上市公司
股东的净利润为 823.20 亿元，同比下降 4.53%；经营活动现金流量净额为 615.22 亿元。
合并利润表中营业总收入为 1,720.54 亿元，包含主业收入和利息收入。

适合验证的能力点：

- 区分“营业收入”和“营业总收入”，避免把财务公司利息收入错误并入主业收入。
- 抽取季度数据，并与全年数做加总校验。
- 处理高分红、回购、每股收益和加权平均净资产收益率等指标。
- 在表格跨页时保留科目、年度、单位和所有权归属关系。

### 招商银行

公开披露资料显示，招商银行 2025 年实现营业收入约 3,375.32 亿元，归属于本行股东的净利润
约 1,501.81 亿元；净利息收入约 2,155.93 亿元，非利息净收入约 1,219.39 亿元。银行年报还包含
不良贷款率、拨备覆盖率、核心一级资本充足率等监管口径指标。

适合验证的能力点：

- 识别银行专用科目，不套用制造业利润表假设。
- 保留“人民币百万元”与“亿元”之间的换算痕迹。
- 校验净利息收入、非利息净收入、营业收入之间的构成关系。
- 对不良贷款率、拨备覆盖率、资本充足率等比例指标保留百分号和口径说明。

## 全量验证结果

本地 macOS CPU 环境已完成一次全量验证。MinerU 使用本地 ModelScope pipeline 模型缓存，
`MINERU_MAX_CONCURRENCY=1`，避免多份超长 PDF 同时解析造成内存峰值。

4 份 PDF 均全量解析成功，行级证据覆盖均为 100%。招商银行 warning 较多，主要因为银行年报中
电话号码、监管比例、附注表和多个金额合并单元格密集出现；当前策略会保留 warning，同时不把这些
软风险计入硬失败。

## 建议执行方式

1. 本地 CPU 可直接全量跑中文年报，但建议串行执行。
2. GPU 或远端 MinerU 服务主要用于加速，不改变结构化目标和质量口径。
3. 用 evaluator 检查关键指标、单位换算、硬性数值一致性和 warning 数量。
4. 样本目录保留 PDF、公开 URL、SHA256 和分析文档，保证输入可复核。

## 期望补充到结果报告的字段

| 字段 | 说明 |
| --- | --- |
| `source_url` | 官方披露入口或交易所公告链接。 |
| `source_sha256` | 下载后本地计算，用于证明输入未变。 |
| `page_span` | 实际解析页段；全量解析时记录为 `all`。 |
| `key_metrics` | 营收、净利润、现金流、分部收入、行业专用指标。 |
| `unit_normalization` | 原始单位与目标单位的映射。 |
| `consistency_checks` | 加总、同比、比例和跨页一致性检查结果。 |
| `warnings` | 合并单元格、OCR、口径差异、四舍五入等风险提示。 |
